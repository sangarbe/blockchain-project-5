# Udacity Blockchain Capstone

The capstone will build upon the knowledge you have gained in the course in order to build a decentralized housing product.

## Requirements and versions used

- Truffle v5.5.6 (core: 5.5.6)
- Ganache v^7.0.3
- Solidity v0.5.16 (solc-js)
- Node v16.15.0
- Web3.js v1.5.3

## Run tests

Execute the following commands in indicated order:

```bash
// 1. Clone the repository
git clone git@github.com:sangarbe/blockchain-project-5.git

// 2. Navigate to repository folder
cd blockchain-project-5

// 3. Install dependencies
npm install

// 4. Navigate to contracts folder
cd eth-contracts

// 5. Run the tests
truffle test
```
